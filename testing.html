Aura Chatbot Widget - Implementation Examples

1. Basic Implementation

Minimal setup with default configuration:

<script src="https://your-cdn.com/aura-chatbot-widget.min.js"></script> <script> AuraChatbotWidget.init({ accountId: 'CLIENT123' }); </script>

2. Custom Branding

Widget with custom colors, logo, and positioning:

<script src="https://your-cdn.com/aura-chatbot-widget.min.js"></script> <script> AuraChatbotWidget.init({ accountId: 'CLIENT456', primaryColor: '#059669', position: 'bottom-left', branding: { logo: 'https://example.com/logo.png', name: 'Green Corp Support' }, greetingMessage: 'Welcome to Green Corp! How can we help you today?' }); </script>

3. Advanced Configuration

Full configuration with custom suggestions and API endpoint:

<script src="https://your-cdn.com/aura-chatbot-widget.min.js"></script> <script> const widget = AuraChatbotWidget.init({ instanceId: 'advanced-widget', accountId: 'CLIENT789', primaryColor: '#7c3aed', position: 'top-right', branding: { logo: 'data:image/svg+xml;base64,PHN2Zy4uLg==', name: 'TechCorp AI' }, greetingMessage: 'Hi! I\'m TechCorp\'s AI assistant. Ready to help!', suggestions: [ 'Tell me about your products', 'How do I get started?', 'What\'s your pricing?', 'Contact sales team', 'Technical support' ], apiUrl: 'https://custom-api.techcorp.com/chat' }); // Event listeners widget.on('opened', () => console.log('Chat opened')); widget.on('message', (data) => console.log('Message:', data)); </script>

4. Programmatic Control

Control the widget programmatically:

// Get widget instance const widget = AuraChatbotWidget.getInstance('default'); // Open chat programmatically widget.open(); // Send a message widget.sendProgrammaticMessage('Hello from JavaScript!'); // Clear conversation widget.clearHistory(); // Update configuration widget.updateConfig({ primaryColor: '#ef4444' }); // Destroy widget widget.destroy();

5. Multiple Instances

Run multiple widget instances on the same page:

// Sales widget AuraChatbotWidget.init({ instanceId: 'sales-widget', accountId: 'SALES001', primaryColor: '#3b82f6', position: 'bottom-right', branding: { name: 'Sales Support' } }); // Technical support widget AuraChatbotWidget.init({ instanceId: 'tech-widget', accountId: 'TECH001', primaryColor: '#059669', position: 'bottom-left', branding: { name: 'Tech Support' } });
